== Description ==
Generates DSKKAR00.dbf and DSKWOR00.dbf files (CSV-like, cp1256 encoded) from a front-end shortcode form and packages them in a ZIP download.

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate "Tamin Exporter".
3. Add the shortcode [tamin_export] to any page.
